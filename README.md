# Sistema de Banco de Dados - Biblioteca

Projeto acadêmico desenvolvido para praticar modelagem de banco de dados.

## Tecnologias
- MySQL
- MySQL Workbench

## Estrutura

O sistema contém as seguintes entidades:

- Alunos
- Autores
- Livros
- Empréstimos

Também foi criada uma tabela intermediária para representar o relacionamento N:N entre livros e autores.

## Funcionalidades do banco

- Cadastro de alunos  
- Cadastro de livros  
- Cadastro de autores  
- Registro de empréstimos  
- Controle de devolução
